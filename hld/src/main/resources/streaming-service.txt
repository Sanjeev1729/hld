Functional Requirements :

- We should be able to upload the movie.

- User should be able to search for a movie.

- User should be able to play the movie.

Non Functional :

- Support different devices & different resolutions

- CAP theorem : Availability is more important then consistency

- Low Latency

- Scalable, reliable, fault tolerant

Scale :

- DAU = 1M

- Read heavy system

- Storage : 1000 of TB
